# SOC Security Dashboard

A cybersecurity dashboard built using Python, Pandas, Streamlit, and Matplotlib.

## Features
- Security event monitoring
- High/Critical alert tracking
- Suspicious IP detection
- Event severity visualization
- Pie chart analytics
- Custom log file upload

## Tools Used
- Python
- Streamlit
- Pandas
- Matplotlib

## Sample Events
- Failed Login
- Malware Alert
- Brute Force Attempt
- DDoS Attack
- SQL Injection
- Phishing Detection
